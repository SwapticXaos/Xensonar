# Machine Room Shift Plan v3 — Scoped Post-FX Depth

Dieser Schritt vertieft die Verschiebung im Bereich **III.1 Myzel / Mastering Engine Room**, ohne die Kern-App zu destabilisieren.

## Was neu wirklich verdrahtet ist

- **Partikel**, **Drone** und **Wellenstarter** speisen nun zusätzlich eine eigene Myzel-Parallelstufe.
- **Forge-Material** wird erstmals als eigener nachgelagerter Post-FX-Pfad mitgenommen, statt nur begrifflich erwähnt zu werden.
- **Myzel selbst** kann ebenfalls gruppenspezifisch in dieselbe Parallelstufe zurückgefärbt werden.

## Warum das driftarm ist

Die Primärwege der Kern-App bleiben bestehen.
Myzel kappt keine Erzeugungswege und übernimmt keine Segmentarbeit zurück.
Stattdessen entsteht eine additive, nachgelagerte Parallelstufe:

1. Forge erzeugt Segmentmaterial.
2. Xensonar performt und spielt.
3. Myzel formt danach gruppenbezogen mit.

## Balance-Regel

Die neue Routing-Tiefe darf die gewachsene Balance nicht zerstören. Deshalb gilt:

- keine harte Insert-Übernahme in Room V,
- keine Rückverlagerung von Loop-/Groove-/Bass-Bau nach Myzel,
- keine implizite Monolith-Renaissance durch versteckte Spezialpfade.

## Konsequenz für die nächsten Schritte

- Forge 2 kann jetzt stärker im Sinne von Segment- und Materiallogik wachsen,
- weil Myzel die nachgelagerte gruppenspezifische Formung bereits sichtbar und hörbar zu tragen beginnt,
- ohne dass man später alles im Tunnelblick nachziehen muss.
